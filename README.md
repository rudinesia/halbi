# Starter Template

This template contain:
- jquery 3.4.1
- popper.js
- bootstrap 4.3.1
- fancybox 3.5.7
- slick carousel 1.8.1
- font awesome 4.7

How to run:
- npm install
- npm run watch (development)
- npm run production (minify js & css)
